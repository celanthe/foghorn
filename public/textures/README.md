# Textures Directory

This directory contains organic image-based textures for the weathered nautical design.

## Required Files

Place the following texture files in this directory:

1. **paper-grain.jpg** (512x512px, ~5-15KB)
   - Subtle aged parchment texture
   - Used on: backgrounds, cards, detail boxes

2. **wood-grain.jpg** (256x256px, ~10-20KB)
   - Weathered coastal wood texture
   - Used on: header, buttons

3. **rope-texture.jpg** (128x128px, ~10-15KB)
   - Natural hemp/sisal rope texture
   - Used on: header accent

## Download Instructions

See `/TEXTURE_DOWNLOADS.md` for detailed download links and optimization instructions.

**Quick start:**
- Paper: https://www.toptal.com/designers/subtlepatterns/ → "Paper"
- Wood: https://ambientcg.com/view?id=Wood049 → 1K Color map
- Rope: https://ambientcg.com/view?id=Rope001 → 512 Color map
